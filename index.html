SquareDay organizes and visualizes your plans using data from the FourSquare and Mapbox APIs. Each fully customizable itinerary contains information about where you'll be and when you'll be there, allowing you to focus on what you'll be doing there.

SquareDay

Squareday is a FourSquare based schedule generator, allowing users to create itineraries with venues and times. This allows the user to have a basic schedule view as well as a calendar representation of all the places the user wants to visit according to their itinerary.

Squareday offers the functionality of a location based schedule creator that helps a user visualize their schedule coordinates in addition to just displaying when different events occur. This visualization helps organize the day by time and location.

SquareDay

    Vocab Lesson
  • SquareDay: A customizable itinerary displayed in time order and in a representatitive map.
  • Venue: Any location available from FourSquare can be added to your itinerary as a venue. The user can add time information and a description to each location

    How to use Square Day:

    First you'll want to create a new SquareDay!
  • Create a new SquareDay by going to the home page, clicking on the My SquareDays dropdown menu and selecting "Add a New Square Day".
  • You will be asked to give the SquareDay a name. Type a name in the dialog and press "Add".


    • Now you have your first SquareDay! Time to add some venues.
  • To add a venue click "Add New Venue" on the right-hand side of the screen.
  • Fill in "Where are you?" and "What are you looking for?" according to your interests. For an example, try "New York" and "Bagels" respectively
  • You'll see Murray's Bagels is one of the first results. Great bagel shop, but they don't toast. Click on your choice of venue from the 5 results listed and set the time attributes accordingly
  • Add a description if you'd like, for example "Gotta eat a bagel!" and press the "Save" button at the bottom of the pop-up.
  • Now you'll see that Murray's (or your choice of bagels) is listed in your SquareDay and a pin has been dropped on your map.
  • You can edit each venue as you please! They will automatically be organized by time.
